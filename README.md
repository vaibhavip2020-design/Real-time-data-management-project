# Real-time-data-management-project
Built real-time AWS data pipeline using S3, Lambda and DynamoDB for automated event-driven processing.

## Objective
Build a real-time AWS event-driven data pipeline using Amazon S3, AWS Lambda and Amazon DynamoDB.

## AWS Services Used
- Amazon S3
- AWS Lambda
- Amazon DynamoDB

## Project Flow
1. Upload JSON file into S3 bucket
2. Lambda triggers automatically
3. Process uploaded data
4. Store output in DynamoDB

## Files Included
- lambda_function.py
- architecture.md
- sample-data/input.json

## Use Case
This project simulates real-time cloud data ingestion and serverless processing.
